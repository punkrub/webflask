# Web Flask Project

โปรเจกต์เว็บไซต์จัดการเมนูร้านกาแฟ พัฒนาด้วย Flask และใช้ Tailwind CSS

## ฟีเจอร์
- มีหน้าเว็บทั้งหมด 10 หน้า
- ใช้ SQLite ในการบันทึกข้อมูลเมนู (ชื่อและราคา)
- ใช้ Tailwind CSS ผ่าน CDN

## วิธีการรันระบบ
1. ติดตั้ง Flask

pip install Flask

2. รันเซิร์ฟเวอร์

python app.py

3. เปิด Web Browser ไปที่ http://127.0.0.1:5000

## คำอธิบายเว็บไซต์
โปรเจกต์นี้คือเว็บไซต์สำหรับร้านกาแฟจำลอง พัฒนาขึ้นด้วยภาษา Python ผ่านเฟรมเวิร์ก Flask เว็บไซต์ถูกออกแบบมาเพื่อแสดงข้อมูลพื้นฐานของร้านและมีระบบสำหรับเพิ่มเมนูเครื่องดื่มและราคา

โครงสร้างของเว็บประกอบด้วยหน้าหลักทั้งหมด 10 หน้า ได้แก่:
1. **Home (`/`)**: หน้าหลัก แสดงรายการเมนูเครื่องดื่มและราคาที่ดึงมาจากฐานข้อมูล
2. **Add Menu (`/add`)**: หน้าแบบฟอร์มสำหรับกรอกชื่อเมนูและราคาเพื่อบันทึกลงฐานข้อมูล SQLite
3. **About (`/about`)**: ข้อมูลเกี่ยวกับร้านกาแฟ
4. **Contact (`/contact`)**: ช่องทางการติดต่อร้าน
5. **Location (`/location`)**: แผนที่และที่ตั้งของร้าน
6. **FAQ (`/faq`)**: คำถามที่พบบ่อย
7. **Gallery (`/gallery`)**: รวมรูปภาพบรรยากาศภายในร้าน
8. **Promotions (`/promotions`)**: ข่าวสารและโปรโมชั่น
9. **Events (`/events`)**: กิจกรรมพิเศษที่จัดขึ้น
10. **Reviews (`/reviews`)**: รีวิวและความคิดเห็นจากลูกค้า
